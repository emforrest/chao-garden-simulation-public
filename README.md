# chao-garden-simulation
## A fun project recreating the Chao Garden from the Sonic Adventure games in a text-based Java simulation
A fan project developed in the summer after A-levels. The Sonic Adventure games feature a section called the Chao Garden, where you hatch and raise creatures called Chao with individual colours and stats. I personally never owned these games, but would watch my cousin play them and create my own Chao sometimes. I thought it would be fun to create a text-based simulation of the Chao Garden to keep up my programming skills during the summer.

The project relies heavily on a database hosted on a remote server and editing using SQL commands. I had especially wanted to practise using the remote server, so this was a big plus. Also it was good to revisit SQL since my A-levels and practise using MySQL Workbench effectively.

This is a work in progress; I am adding features from the original game and my own new features one at a time, and this may be evidenced by some unfinished elements in the current version.

*Note - I believe the database features do not currently work in this version.*
